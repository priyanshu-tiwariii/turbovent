## Getting Started  

Clone the repository:  
```sh
npm i -g turbovent
```  

Install dependencies using **pnpm**:  
```sh
turbovent
```  

Run the development server:  
```sh
cd <projectname>
```  

For a production build:  
```sh
pnpm install
```  

Start the production server:  
```sh
pnpm run dev
```
